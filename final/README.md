# Family Adventures in America's National Parks

**Tagline:** Simple trip planning for families who love the outdoors.

## Project Overview
This website helps families plan trips to U.S. National Parks by providing:
- Park comparisons by region and features
- Kid-friendly trail information with difficulty ratings
- Simple 2-3 day itineraries
- Practical planning tips and safety reminders

## Site Structure
- **index.html** - Homepage with featured parks and filtering
- **park.html** - Dynamic park detail pages
- **itineraries.html** - Sample trip itineraries
- **tips.html** - Planning tips for families

## Design System

### Colors
- **Evergreen Ridge** (#0B3D2E) - Header, navigation, footer
- **Canyon Sunrise** (#C47E36) - Buttons, links, accents
- **Alpine Sky** (#BBD7EA) - Highlight panels, cards
- **Warm Sandstone** (#F5F2E8) - Page background

### Typography
- **Headings:** Bitter (slab-serif)
- **Body:** Source Sans 3 (sans-serif)

## Features
✅ Responsive design (mobile-first)
✅ Region-based filtering
✅ Sort by name, region, or season
✅ Hamburger menu for mobile
✅ Accessible markup (ARIA labels)
✅ Trail difficulty and safety information
✅ Sample itineraries

## User Scenarios Addressed
1. **Compare parks** - Filter by region, sort by various criteria, view key stats
2. **Simple itineraries** - Pre-planned 2-3 day trips with timing and activities
3. **Trail information** - Difficulty, distance, elevation, and safety tips
